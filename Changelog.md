# 🇬🇧 Changelog (English)

All notable changes to this project will be documented in this section.

## [0.3.2] - 2026-03-24

### 🔄 Changed
- **Visual Rework**: Complete visual overhaul of the Pokédex page for a better UI/UX.

---

## [0.3.1] - 2026-03-24

### ✨ Added
- **New Favorites**: Added 3 new "Favorite" Pokémon (Hisuian Zorua, Ceruledge, Alolan Ninetales).
- **Tags System**: Added the **Variants** tag to more efficiently filter Pokémon with alternative forms.
- **New Achievements**: Added 5 new achievements linked to variants and favorites (bringing the total to 75).

### 🔄 Changed
- **Generation Adjustment**: Fixed the generations for regional variants (e.g., Galar/Hisui assigned to Gen 8, Paldea to Gen 9).
- **Improved UX**: The filter reset button now also clears the text search input.

---

## [0.3.0] - 2026-03-24

### ✨ Added
- **Roster Expansion**: Added 7 new custom Pokémon (Friends' favorites: Gengar, Meowth, Bulbasaur, Articuno, Eevee, Umbreon, Nidoqueen).
- **New Tag**: Created the **Favorites** tag.
- **Twitch Integration**: New **Channel Points** reward (50k points) allowing viewers to add their favorite Pokémon to the game if it's not already in the roster!
- **Achievements UI**: The 70 existing achievements are now visible and integrated into the Web interface (more to come).
- **Leaderboards**: Added 4 leaderboards on the website (Total catches, Unique catches, Unlocked achievements, Total achievement points).
- **Pokédex UX**: Added a button to quickly reset all active filters.

### 🔄 Changed
- **UI Overhaul**: Major graphic redesign of the entire website.

---

## [0.2.2] - 2026-03-21

### ✨ Added
- **New 3D Models**: Added 20 new Pokémon with their animated 3D models, including 2 legendaries, 1 starter, and 4 fossils (mostly from Gen 5).

### 🔄 Changed
- **Infrastructure**: Changed hosting provider to ensure a much smoother, faster, and more reliable website.

---

## [0.2.1] - 2026-03-20

### ✨ Added
- **Advanced Filters**: The Pokédex now allows precise sorting by **generation**, **type**, or **tag**.
- **Search Engine**: Implemented a name search bar to find a Pokémon instantly.

---

## [0.2.0] - 2026-03-04

### ✨ Added
- **Achievement System (Bot Side)**: 70 achievements categorized by tiers are now active on the stream. Each achievement grants a specific number of points.
- **Individual Statistics**: The Pokédex now displays the total number of times a specific Pokémon has been caught by the player.

### 🐛 Fixed
- Various background bug resolutions.

---

## [0.1.1] - 2026-01-24

### 🐛 Fixed
- Resolved critical bugs related to user management and authentication.

### 🐛 Fixed
- Optimizations and various bug fixes.
